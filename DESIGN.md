# HidroConvert Design System

## Colors

| Element | Value | Usage |
|----------|-------|-------|
| Canvas Cream | `#F3F0EE` | Main background |
| Ink Black | `#141413` | Text, primary buttons, footer |
| Signal Orange | `#CF4500` | Accents, consent |
| Light Signal Orange | `#F37338` | Decorative indicators |
| Lifted Cream | `#FCFBFA` | Elevated surfaces |
| Slate Gray | `#696969` | Secondary text |

## Border Radius

- Buttons: 20px
- Containers: 40px
- Pills: 999px

## Typography

- Font: Sofia Sans (fallback: Inter, Arial)
- Weights: 450, 500, 700

## Shadows

- Subtle elevation: `rgba(0,0,0,0.04) 0px 4px 24px`

## Component Spacing

- Tight: 8px
- Default: 16px
- Loose: 24px
- Section: 48px
